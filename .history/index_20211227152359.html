My Work

See examples of my work below.

Instapaw is an Instagram clone where users are met with a login/splash page. Signup is possible with passwords being fully encrypted when stored in the database (SQLAlchemy). The back-end utilizes Python, Flask, and the entire application is containerized in a docker container. Front-end technologies include React.js / Redux.js. Front-end makes calls to the back-end API to create, render, update and delete data (CRUD). When a user is logged in, they can like or comment on their friends' recent posts, as well as search for friends as well as follow/unfollow them as they please.
When users post an image, this gets stored on Amazon Web Services (AWS S3), allowing for a full and dynamic user experience. The entirety of the website utilizes cutting edge algorithms and programming best practices to ensure a page refresh is never needed, thus, improving speed, reliability and ultimately, user experience.
Multiple modals are also utilized throughout the website to allow the user to access the website's full potential without having to navigate away from a page they were on.

Rev Up is a mix between Meetup and Eventbrite, but for cars and bikes! The back-end utilizes Node.js, Python, and Flask-SQLAlchemy. The front-end utilizes React.js/Redux.js and makes calls to the back-end API to create, render, update and delete data(CRUD).
This website utilizes WTForms on back-end routes for authentication and blocks non-logged in users from accessing endpoints with the help of it's validators.
On arrival, users must be logged in or create an account to go further.
When a user creates an event or RSVP's an event, the type of event with a link to the posting will be available on the user's calendar for the date it's scheduled. Thanks to React/Redux, as well as the most cutting edge algorithms and programming best practices, the entirety of the website can be utilized to it's full potential without ever needing a page refresh. This ensures speed, reliability and ultimately a great user experience.
